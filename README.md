# Workflow:
Work on Development or Feature branches, use git as usual
On Pull-Request to main branch:
Include #major, #minor or #patch in pull-request title to mark a release This will increment the appropriate decimal in version numbering i.e: 
- #major	1.2.3 → 2.0.0
- #minor	1.2.3 → 1.3.0
- #patch	1.2.3 → 1.2.4
